名称 传入参数 返回值 兼容 注意 是否改变数组
concat n个数组或者非数组 新的Array实例 >= ie6 对象引用会直接复制引用到新数组 否
join (separator) 字符串 ie6 默认连接符：",", 未赋值的，表现为"" 否
every (callback(element, index, arr), thisArg:callback执行时的this值，不指定的话，strict为undefined， 否则为全局) bool值 >=ie9 被删除和从未赋值的元素不会被访问，访问的是调用那一瞬间的快照 否
some (callback(element, index, array){}[,thisArg]) bool值 ie9 同上 否
filter 同上 返回新的Array实例 ie9 同上 否
forEach 同上 返回新的Array实例 ie9 同上+没有办法中止或者跳出 forEach 循环，除了抛出一个异常。 否
map (callback(element, index, arr), thisArg:callback执行时的this值，不指定的话，strict为undefined， 否则为全局) 新的Array实例 ie9 不修改数组本身，不会访问未赋值和删除的元素，访问的是快照 否
indexOf (value, startIndex) value第一次出现的位置 ie9 未找到返回-1，startIndex为负数，表示从个倒数第几个开始 否
pop 无 被删除的元素 ie6 通用，可用call和apply调用，会改变原数组 是
push (elementN,) 新的length ie6 会改变原数组 是
shift 无 被删除的元素 ie6 改变原数组，可被用于对象，如果该对象没有length属性，将添加值为0的length 是
unshift (element1, ..., elementN) 该数组的length ie6 是
reverse 无 颠倒后的数组 ie6 会改变原始数组 是
sort ([compareFunction(a, b)]) 排好序的数组 ie 改变数组，默认Unicode位点排序 是
splice (start,deleteCount[, itemN...]) 被删除的元素组成的数组 ie6 改变数组 是
slice (begin[, end]) 含有提取元素的新数组 ie6 ie8- end不能为空 否
reduce (callback(pre, cur, index, arry){}, initialValue) 函数累计处理的结果 ie9 数组为空，且没有initialValue -> TypeError 否
reduceRight (callback(pre, cur, index, arry){}, initialValue) 函数累计处理的结果 ie9 数组为空，且没有initialValue -> TypeError 否
